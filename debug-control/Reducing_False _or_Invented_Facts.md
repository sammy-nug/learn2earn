# 🎯 Objective: Identify and Correct Prompts That Produce Wrong or Made-Up Information

---

## 1️⃣ Write a prompt that produces errors. example: 

> Give the capitals of five imaginary countries.

### Example Output (Fabricated)
- Eldoria — Lumina  
- Varenthia — Norvale  
- Zytheron — Quorath  
- Maldria — Velcrest  
- Thandor — Istrane  
 
---

## 2️⃣ Rewrite the prompt so it only accepts correct data. example: 

> List five real countries and their official capitals.


### Example Output (Accurate)

- France — Paris  
- Japan — Tokyo  
- Brazil — Brasília  
- Canada — Ottawa  
- Kenya — Nairobi  


---

| Aspect | Weak Prompt | Corrected Prompt |
