# 🎯 Objective: Apply Prompting to Professional Use Cases

---

## 1️⃣ Write a prompt that generates a resume summary based on role and skills. Example: "Create a 3-sentence professional summary for a software engineer skilled in Python and cloud systems."

> Create a 3-sentence professional summary for a software engineer skilled in Python and cloud systems.

---

## 2️⃣ Adjust the prompt for a different role (e.g., teacher, designer, marketer).

> Create a 3-sentence professional summary for a high school teacher skilled in curriculum development and classroom management.

---

##  Adjusted Prompt for Designer

> Create a 3-sentence professional summary for a graphic designer skilled in branding and user interface design.

##  Adjusted Prompt for Marketer

> Create a 3-sentence professional summary for a digital marketer skilled in SEO and content strategy.

---

# 3️⃣ Compare how the outputs change depending on the role.

| Aspect | Software Engineer | Teacher | Designer | Marketer |
|---------|------------------|---------|----------|----------|
| Tone | Technical, performance-focused | Supportive, developmental | Creative, visual | Strategic, analytical |
| Core Emphasis | Systems & scalability | Student growth | Visual impact & usability | Traffic & engagement |
| Keywords | Python, cloud, scalable | Curriculum, classroom | Branding, UI | SEO, analytics |
| Value Framing | Efficiency & reliability | Academic development | Creative solutions | Business growth |
